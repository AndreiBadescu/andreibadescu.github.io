Pentru a introduce o formula propozitionala tastati litere (A-Z) sau caractere specificate mai jos.

Lista conectori logici (tastarea unei cifre de mai jos va insera un conector logic):
  1  = ¬ (negatie)
  2  = ∧ (conjunctie)
  3  = ∨ (disjunctie)
  4  = ⇒ (implicatie)
  5  = ⇔ (echivalenta)
  9* = (
10* = )

Lista comenzi:
DELETE = sterge formula porpozitionala introdusa
ENTER = executa programul
BACKSPACE = sterge ultimul caracter introdus
CTRL-Z = undo
CTRL-V = paste text
